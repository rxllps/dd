## 🚀 Шаги для запуска:

### 1. Создание и применение миграций БД

```bash
# Создать новые миграции на основе моделей
python manage.py makemigrations

# Применить миграции к БД
python manage.py migrate
```

### 2. Создание суперпользователя (администратора)

```bash
python manage.py createsuperuser
```

Введите:
- Email: admin@karaoke.local
- Password: (выберите пароль)

### 3. Создание тестовых данных

Вход в админ-панель: http://localhost:8000/admin

#### Создайте типы комнат (RoomType):
- VIP (премиум)
- Стандарт
- Люкс

#### Создайте комнаты (Room):
Пример:
- Название: "VIP Люкс"
- Вместимость: 10 человек
- Оборудование: "Микрофоны, гитара, колонки"
- Цена: 2000 руб/час
- Типы: выберите VIP
- Видео URL: https://youtube.com/embed/...

#### Создайте временные слоты (TimeSlot):
Пример:
- Комната: "VIP Люкс"
- Дата: 2024-01-15
- Время: 18:00
- Цена: 2000 (за час)
- Доступна: ☑ (отмечено)

### 4. Запуск приложения

```bash
# На Windows
python manage.py runserver

# На Linux/Mac
python manage.py runserver 0.0.0.0:8000
```

Приложение доступно: http://localhost:8000

### 5. Основные страницы:

- **Главная**: http://localhost:8000/ - список доступных комнат
- **Информация**: http://localhost:8000/my_karaoke/ - о караоке-баре
- **Новые комнаты**: http://localhost:8000/upcoming/ - все комнаты
- **Цены**: http://localhost:8000/price/ - цены и скидки
- **Контакты**: http://localhost:8000/contact/ - контакты
- **Гостевая**: http://localhost:8000/guest/ - гостевая книга
- **Администрация**: http://localhost:8000/admin/ - админ-панель

## 📝 Примеры использования:

### Регистрация и вход:
1. Перейти на http://localhost:8000/sign_in
2. Нажать "Регистрация"
3. Заполнить форму (email, пароль)
4. Подтвердить

### Бронирование:
1. На главной странице выбрать дату
2. Выбрать комнату
3. Нажать "Забронировать"
4. Выбрать время (TimeSlot)
5. Заполнить данные и отправить

### Просмотр истории:
1. Войти в аккаунт
2. Нажать "Кабинет"
3. Увидеть все бронирования

## ⚠️ Если есть ошибки:

1. **Проблема с миграциями:**
   ```bash
   # Удалить БД
   rm db.sqlite3
   # Создать заново
   python manage.py migrate
   ```

2. **Проблема с импортами:**
   - Убедитесь, что все приложения в INSTALLED_APPS в `settings.py`

3. **Проблема с медиа:**
   - Убедитесь, что папка `media/` существует и доступна
