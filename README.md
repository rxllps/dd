# Система онлайн-бронирования караоке-комнат

Веб-приложение для развлекательного клуба: бронирование комнат по временным слотам, пакеты услуг, онлайн-оплата (тестовый режим / заготовка под ЮKassa), личный кабинет и отчёты для администратора.

**Стек:** Python, Django 1.9, SQLite, HTML/CSS, jQuery.

## Возможности

- **Комнаты и категории** — вместимость, оборудование, цена за час, расписание работы.
- **Временные слоты** — статусы: свободен, зарезервирован, оплачен, отменён; генерация слотов командой.
- **Пакеты услуг** — фиксированная цена, длительность, скидки, привязка к типам комнат.
- **Расчёт цены** — выходные +20%, групповая скидка от 5 часов −15%, пакетные скидки.
- **Оплата** — mock-провайдер для разработки; настройка `PAYMENT_PROVIDER=yookassa` для продакшена.
- **Отмена** — бесплатно за 2+ часа до начала, иначе штраф 50%.
- **Админка Django** — управление комнатами, слотами, пакетами, платежами, отчёты PDF.

## Быстрый старт

```bash
cd Documents/GitHub/Cinema
pip install django==1.9.5 reportlab pillow
python manage.py migrate
python manage.py seed_packages
python manage.py generate_timeslots --days 14
python manage.py createsuperuser
python manage.py runserver
```

Сайт: http://127.0.0.1:8000/  
Админка: http://127.0.0.1:8000/admin/

## Основные URL

| URL | Назначение |
|-----|------------|
| `/` | Расписание комнат по дням |
| `/room/<url_name>/` | Слоты комнаты |
| `/booking/slot/slot_id=<id>/` | Окно бронирования |
| `/payment/<id>/` | Страница оплаты |
| `/price/` | Пакеты и тарифы |
| `/cabinet/` | Личный кабинет |

## Платежи

В `untitled/settings.py`:

```python
PAYMENT_PROVIDER = 'mock'  # или 'yookassa'
YOOKASSA_SHOP_ID = 'ваш_shop_id'
YOOKASSA_SECRET_KEY = 'ваш_ключ'
```

Webhook (заглушка): `POST /payment/webhook/` с полями `booking_id`, `external_id`.

## Команды управления

```bash
python manage.py generate_timeslots --days 30 --room vip-room
python manage.py seed_packages
```

## Структура моделей

- `RoomType`, `Room` — комнаты и категории
- `TimeSlot` — интервалы бронирования
- `ServicePackage` — пакеты услуг
- `Booking`, `Reservation` — оплаченные и временные брони
- `Payment` — платёж по бронированию
- `BookingStat` — статистика для отчётов

## Тема диплома

Проект покрывает: моделирование предметной области клуба, веб-интерфейс бронирования, подсистему расчёта стоимости и оплаты, администрирование и отчётность.
